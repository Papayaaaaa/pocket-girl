# 📦 尾款助手 - 二次元/汉服/Lo裙记录小程序

一个帮二次元、汉服、Lo娘记录谷子预购+尾款的小程序。

## 功能

### 核心功能
- ✅ 记录购买：商品名称、类型、店铺、定金、尾款金额、截止日期
- ✅ **📸 图片上传**：可上传订单截图/实物图（最多9张）
- ✅ 分类管理：谷子 / 汉服 / Lo裙 / 其他
- ✅ 尾款提醒：临近截止日期提醒
- ✅ **🔔 服务号推送**：开通微信服务号推送提醒（需配置）
- ✅ 记账统计：累计消费、分类统计
- ✅ 本地存储：数据保存在手机本地

### 新增功能 (v1.1)
- ✅ **🔍 搜索功能**：支持按商品名称/店铺/分类搜索
- ✅ **📊 首页汇总**：顶部显示待尾款数量、金额、已完成数、逾期数
- ✅ **⚠️ 逾期提醒**：已逾期订单红色高亮显示
- ✅ **💳 快捷支付**：首页一键标记尾款已付
- ✅ **📋 批量操作**：批量选择删除记录
- ✅ **📤 数据导出/导入**：支持备份和恢复数据
- ✅ **📈 年度统计**：查看年度消费趋势
- ✅ **📅 月度统计**：查看月度消费明细
- ✅ **🔔 自定义提醒**：可设置多个提前提醒天数
- ✅ **✏️ 编辑时删除图片**：添加/编辑时可删除已上传图片
- ✅ **📲 分享功能**：支持分享给朋友

## 部署步骤

### 1. 注册小程序

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 选择「小程序」，用身份证注册
3. 完成注册后，登录小程序管理后台

### 2. 获取 AppID

1. 在公众平台左侧点击「开发管理」
2. 点击「开发设置」
3. 找到「AppID」，复制备用

### 3. 配置项目

1. 下载 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开项目目录 `pocket-girl/`
3. 修改 `project.config.json` 中的 `"appid"` 为你的 AppID
4. 如果有图标，把 `icons/` 目录下的图标换成你自己的（推荐尺寸: 81x81px）

### 4. 导入运行

1. 打开微信开发者工具
2. 点击「导入项目」
3. 选择 `pocket-girl` 文件夹
4. 点击编译，即可预览

### 5. 上传发布

1. 在开发者工具中点击「上传」
2. 填写版本号和备注
3. 登录公众平台，提交审核
4. 审核通过后即可发布

## 服务号模板消息配置（可选）

如果你需要微信服务号推送提醒：

1. 有一个认证过的微信服务号
2. 在服务号后台添加模板：
   - 标题：尾款提醒
   - 字段：商品名称、尾款金额、截止日期
3. 需要后端服务器调用微信 API 发送模板消息

## 项目结构

```
pocket-girl/
├── app.js                      # 小程序入口
├── app.json                    # 小程序配置
├── app.wxss                    # 全局样式
├── pages/
│   ├── index/                  # 首页（订单列表）
│   ├── add/                    # 添加记录（含图片上传）
│   ├── detail/                 # 详情/编辑（含图片预览）
│   ├── stats/                  # 记账统计（半月/月度）
│   ├── yearly/                 # 年度统计
│   ├── monthly/                # 月度详情
│   └── settings/               # 服务号推送设置
├── cloudfunctions/
│   ├── sendReminder/           # 云函数（发送提醒）
│   └── aiRecognize/            # 云函数（AI识别）
└── icons/                      # TabBar 图标
```

## 技术栈

- 微信小程序原生开发
- 本地存储（wx.setStorage）
- 微信服务号模板消息（可选）
- AI 智能识别（需配置 API Key）
