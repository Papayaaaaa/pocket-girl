## 股票播报脚本
- 路径: `/root/.openclaw/workspace/scripts/stock_broadcast_notify.py`
- 数据源: 东方财富 API
- 输出格式: 按用户指定格式（📊 每日股市播报）

## 已知问题
- OpenClaw cron 的 isolated session 模式有超时 bug，任务会卡住超时
- 解决方案：使用 system cron 作为备用方案，或手动推送

## 定时任务配置
| 任务 | 时间 | 状态 |
|------|------|------|
| morning-news-broadcast | 每天 9:00 | error (超时) |
| morning-stock-broadcast | 工作日 11:30 | error (超时) |
| afternoon-stock-broadcast | 工作日 15:00 | error (超时) |
