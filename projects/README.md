# 项目结构说明

```
/workspace/projects/
├── kunming-housing/     # 昆明房产分析项目
│   ├── docs/            # 文档资料
│   ├── data/            # 原始数据
│   ├── scripts/         # 脚本代码
│   └── reports/         # 分析报告
│
├── stock-broadcast/      # 股票播报项目
│   ├── docs/
│   ├── data/
│   ├── scripts/
│   └── reports/
│
└── news-broadcast/      # 早报播报项目
    ├── docs/
    ├── data/
    ├── scripts/
    └── reports/
```

## 目录用途

- **docs/**: 项目文档、API说明、配置说明
- **data/**: 原始数据、缓存数据
- **scripts/**: 可执行脚本、代码
- **reports/**: 分析报告、输出结果
